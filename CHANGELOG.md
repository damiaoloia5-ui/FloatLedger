# Changelog

## 1.0.0 - 2026-06-14

- Initial Windows installer release as "DeepSeek Monitor".
- Desktop balance overlay for AI API accounts (DeepSeek supported).
- Configurable refresh interval, theme, opacity, tray icon, and auto-start.
- Multi-language UI support (7 languages).
- API Key is entered by each user at runtime and stored locally in the user's AppData directory.
- Renamed to FloatLedger to reflect multi-vendor support.
